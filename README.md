# Customers backend
RESTful API to make CRUD operations for Customers

## Stack

- Java 11
- Spring Boot
- Spring Security
- Spring Data
- JWT Token
- H2 in memory database
- JUnit
- Mockito
- Lombok
- Mapstruct
- Maven

## How to run

1. Generate resources running: 
```sh mvn clean install ```
1. Run the fat jar:
```sh java -jar target/customers-backend.jar ```
